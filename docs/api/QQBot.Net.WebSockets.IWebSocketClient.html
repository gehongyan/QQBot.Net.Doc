BinaryMessage

当接收到二进制消息时触发。

Event Type

Func<byte[], int, int, Task>

Remarks

事件参数：

  1. byte[] 参数是接收到的二进制消息的字节数组。
  2. int 参数是接收到的二进制消息的字节数组的起始位置。
  3. int 参数是接收到的二进制消息的字节数组的长度。
