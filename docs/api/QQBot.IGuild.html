Remarks

note

此属性仅对基于网关连接的客户端有意义。

此属性为 true 表示，此服务器实体已完整缓存基础数据，并与网关同步。
缓存基础数据包括服务器基本信息、频道、角色、频道权限重写、当前用户在服务器内的昵称。
